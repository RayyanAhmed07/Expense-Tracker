# 💸 Expense Tracker

A simple command-line Python app that lets you track your expenses.  
Perfect beginner project that stores your data in a JSON file — no databases required!

---

## ✅ Features

- Add a new expense (category + amount)
- View all saved expenses
- See total amount spent
- Stores all data in a local file (`expenses.json`)

---

## ▶️ How to Run

1. Make sure you have Python installed
2. Open a terminal in the project folder
3. Run the app using:

```bash
python tracker.py
